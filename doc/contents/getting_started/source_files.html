Source code if you sync from Mercurial repository includes SALVIA code and some third-party source code.

Following directories are in ${HOME} directory:

  • 3rd_party
    • llvm: Source code of LLVM
    • wtl: Source code of WTL
    • Thread pool: Source code of thread pool, which is a candidate of Boost library.
  • bin
    • resource: Resource files, includs models and textures.
    • ${PLATFORM}_${COMPILER}_${CONFIG}: built executable binaries and dynamic/shared libraries.
